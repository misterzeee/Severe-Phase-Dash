# Severe-Phase-Dash

```lua
bytecode = game:HttpGet("https://github.com/misterzeee/Severe-Phase-Dash/raw/refs/heads/main/dash.lua")
local func = luau.load(bytecode)
func()
```

## Using Custom Values

```lua
bytecode = game:HttpGet("https://github.com/misterzeee/Severe-Phase-Dash/raw/refs/heads/main/dash.lua")
local func = luau.load(bytecode)
func()
DashDistance  = 20
DashCooldown  = 0.5
DashKey       = "R"
```
